Privacy Policy – Page Summarizer Extension

Effective Date: July 5, 2025

1. Introduction

This extension, Page Summarizer, respects your privacy. This policy outlines what data we access, how we use it, and how we protect it.

2. Information We Collect

  • Website Content: When you click the “Summarize This Page” button, the extension temporarily reads the visible text content of the current web page for the sole purpose of summarization.
  • User-Provided Text: If you paste text into the input box, that text is processed for summarization.

We do NOT collect or store any of the following:

  • Personally identifiable information (name, email, address, etc.)
  • Health, financial, authentication, or communication data
  • Browsing history, location data, or sensitive information

3. How We Use Information

The extension uses content only to generate summaries and key definitions using an AI API (Groq). Nothing is stored or shared.

4. Data Retention

No data is stored by the extension or backend server. Processing is temporary.

5. Third Parties

We use Groq AI to generate summaries. Text is sent securely and is not linked to any user or stored by the extension.

6. Security

We use HTTPS for all API communication to ensure your data is encrypted in transit.

7. Consent

By using this extension, you agree to this privacy policy.
